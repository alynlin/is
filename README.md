======

一些常用的验证函数：

```go
// 判断是否为数值
is.Number("123")

// 判断是否为ISBN序列号
is.ISBN("1-919876-03-0")

// 身份证验证
is.GB11643("33232333211233432")

// 银行卡验证
is.BankCard("33232123234")
```

安装
----

```shell
go get github.com/issue9/is
```
